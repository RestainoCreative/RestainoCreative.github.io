# Justin Restaino — Portfolio

Personal portfolio site for Justin Restaino — Creative Director / Creative Producer.

Live: https://restainocreative.github.io/

## Pages
- `index.html` — home (hero reel, statement, by-the-numbers, what I do, contact)
- `work.html` — selected work index (stacked full-bleed reel)
- `project.html` — project case study template ("Big Screen")

## Stack
Static site — React 18 + Babel transpiled in the browser (no build step). Styles in `styles.css`.

## Local preview
```bash
python3 -m http.server 8000
# then open http://localhost:8000/
```
